The purpose of this script is to automate a user interacting with a discord bot called Mudae (https://mudae.fandom.com/wiki/Mudae).
This script uses a mixture of api requests and simulated keypresses and clicks to interact with the bot.

More information about the bot and the specific interactions that the script does: https://mudae.fandom.com/wiki/List_of_Commands#.24waifu
